Example 1.2.32

Let \( A \) and \( B \) be \( n \times n \) matrices. We say that \( A \) is similar to \( B \), denoted \( A \sim B \), if there exists a nonsingular \( n \times n \) matrix \( P \) such that

\( A = P B P^{-1} \).

This is an equivalence relation on \( n \times n \) matrices because:

  1. (Reflexive property) \( A = I_n A I_n^{-1} \), where \( I_n \) is the \( n \times n \) identity matrix, so \( A \sim A \).
  2. (Symmetric property) If \( A \sim B \), then \( A = P B P^{-1} \) for some invertible \( P \). Hence, \( B = P^{-1} A (P^{-1})^{-1} \), so \( B \sim A \).
  3. (Transitive property) If \( A \sim B \) and \( B \sim C \), then there exist invertible matrices \( P \) and \( Q \) such that \( A = P B P^{-1} \) and \( B = Q C Q^{-1} \). Thus, \( A = (PQ) C (PQ)^{-1} \) and \( A \sim C \).

Similar matrices share many linear algebraic properties, for example, they have the same determinants, traces, and eigenvalues. The Jordan Canonical Form is often chosen to represent these similarity classes.

Example 1.2.33

Let \( f, g: \mathbb{R} \to \mathbb{R} \) be two differentiable functions. Define \( f \sim g \) if and only if \( f' = g' \). This is an equivalence relation on the set of differentiable real-valued functions. From calculus, we know that \( f(x) = g(x) + C \), and typically the antiderivative with \( C = 0 \) is chosen as the representative of the equivalence class.
