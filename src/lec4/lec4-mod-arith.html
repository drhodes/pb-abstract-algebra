Definition 1.2.37

Let \( r \) and \( s \) be two integers and let \( n \) be a positive integer. We say that \( r \) is congruent to \( s \) modulo \( n \), denoted \( r \equiv s \ (\text{mod } n) \), if and only if \( r - s = kn \) for some \( k \in \mathbb{Z} \).

Example 1.2.38

Compute the following residues:

  1. \( 10 \equiv 1 \ (\text{mod } 3) \) since \( 10 - 1 = 9 = 3 \cdot 3 \).
  2. \( 15 \equiv 0 \ (\text{mod } 5) \) since \( 15 - 0 = 15 = 3 \cdot 5 \).
  3. \( 15 \not\equiv 0 \ (\text{mod } 7) \) since \( 15 - 0 \) is not a multiple of 7.
  4. \( 30 \equiv 48 \ (\text{mod } 9) \) since \( 30 - 48 = -18 = -2 \cdot 9 \).
  5. \( 23 \equiv 2 \ (\text{mod } 7) \) since \( 23 - 2 = 21 = 3 \cdot 7 \).

Proposition 1.2.39

Congruence modulo \( n \) is an equivalence relation on \( \mathbb{Z} \) for any positive integer \( n \).

Proof

Let \( r, s, t \in \mathbb{Z} \). Then:

  1. (Reflexive property) \( r - r = 0 = 0 \cdot n \), so \( r \equiv r \ (\text{mod } n) \).
  2. (Symmetric property) If \( r \equiv s \ (\text{mod } n) \), then \( r - s = kn \) for some \( k \in \mathbb{Z} \). It follows that \( s - r = (-k)n \), hence \( s \equiv r \ (\text{mod } n) \).
  3. (Transitive property) If \( r \equiv s \ (\text{mod } n) \) and \( s \equiv t \ (\text{mod } n) \), then \( r - s = kn \) and \( s - t = \ell n \) for some \( k, \ell \in \mathbb{Z} \). Adding these gives \( r - t = (k+\ell)n \), so \( r \equiv t \ (\text{mod } n) \).

Congruence Classes Modulo 3

Considering the equivalence relation on the integers modulo 3, we have:

  • \( [0] = \{ \dots, -3, 0, 3, 6, \dots \} \)
  • \( [1] = \{ \dots, -2, 1, 4, 7, \dots \} \)
  • \( [2] = \{ \dots, -1, 2, 5, 8, \dots \} \)

Notice that \( [0] \cup [1] \cup [2] = \mathbb{Z} \) and the sets are disjoint. Thus, they form a partition of \( \mathbb{Z} \). In general, there will be \( n \) congruence classes modulo \( n \), and each class can be represented by a unique natural number \( x \) such that \( 0 \leq x < n \). In fact, this representative \( x \) is just the remainder when the integer is divided by \( n \).

The set of all congruence classes modulo \( n \) is denoted \( \mathbb{Z}_n \) and contains \( n \) elements.
