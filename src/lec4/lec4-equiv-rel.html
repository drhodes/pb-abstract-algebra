Lecture 4: Equivalence Classes

The worst form of inequality is to try to make unequal things equal.

1.2. Sets and Equivalence Relations

Definition 1.2.30

An equivalence relation \( \sim \) on a set \( X \) is a relation \( \sim \subseteq X \times X \) such that:

  1. (Reflexive property) \( x \sim x \) for all \( x \in X \).
  2. (Symmetric property) \( x \sim y \) implies \( y \sim x \).
  3. (Transitive property) If \( x \sim y \) and \( y \sim z \), then \( x \sim z \).

For any \( x \in X \), define the set \( [x] = \{ y \in X \mid x \sim y \} \). This set is called the equivalence class of \( x \) and each element of this set is called a representative of the equivalence class.

Example 1.2.31

Let \( X = \{ (a,b) \mid a,b \in \mathbb{Z},\, b \neq 0 \} \) and define a relation \( \sim \) on \( X \) by

\( (a,b) \sim (c,d) \) if and only if \( ad = bc \).

This relation is an equivalence relation on \( X \) because:

  1. (Reflexive property) Since \( ab = ab \), we have \( (a,b) \sim (a,b) \).
  2. (Symmetric property) If \( (a,b) \sim (c,d) \) then \( ad = bc \), which implies \( cb = da \) and so \( (c,d) \sim (a,b) \).
  3. (Transitive property) If \( (a,b) \sim (c,d) \) and \( (c,d) \sim (e,f) \), then \( ad = bc \) and \( cf = de \). Multiplying the first equation by \( f \) and the second by \( b \) yields \( adf = bcf = bde \). Since \( d \neq 0 \), dividing by \( d \) gives \( af = be \), so \( (a,b) \sim (e,f) \).

Thus, the relation is an equivalence relation. The set of equivalence classes, denoted \( \mathbb{Q} = \{[x] \mid x \in X\} \), represents the rational numbers, with the ratio in lowest terms typically chosen as the representative for each class.
