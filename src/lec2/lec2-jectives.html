Definition 1.2.19

Let `f: A \to B` be a function. Let `a_1, a_2 \in A`. We say that `f` is injective or one‑to‑one if `f(a_1) = f(a_2)` implies that `a_1 = a_2`; in other words, for each `b \in f(A)` there is a unique `a \in A` such that `f(a) = b`.

We say that `f` is surjective or onto if the image of `f` equals the codomain of `f` (`f(A) = B`), or in other words, for all `b \in B` there exists an `a \in A` such that `f(a) = b`.

If `f` is both injective and surjective, then we say that `f` is bijective.

Example 1.2.20

If `f: \mathbb{R} \to \mathbb{R}`, `g: \mathbb{R} \to \mathbb{R}`, and `h: \mathbb{R} \to \mathbb{R}` are given by `f(x) = x^3`, `g(x) = e^x`, and `h(x) = x^2`, then `f` is bijective, `g` is injective but not surjective, and `h` is neither injective nor surjective. On the other hand, the mappings `g_1: \mathbb{R} \to (0, \infty)` and `h_1: \mathbb{R} \to [0, \infty)` given by `g_1(x) = e^x` and `h_2(x) = x^2` are both surjective maps but only `g_1` is injective. If we consider `h_2: [0, \infty) \to [0, \infty)` given by `h_2(x) = x^2`, then `h_2` is bijective. For these reasons, the convention for defining functions in calculus is very imprecise for higher mathematics.
