Example 3.2.8.

Solve the equation `2x + 1 \equiv 5 \ (\text{mod } 7)` for `x`.

To begin we apply the additive inverse of `1` to both sides of the equation:

(2x + 1) + (-1) &\equiv 5 + (-1) \ (\text{mod } 7) \\ 2x + (1 + (-1)) &\equiv 4 \ (\text{mod } 7) \\ 2x + 0 &\equiv 4 \ (\text{mod } 7) \\ 2x &\equiv 4 \ (\text{mod } 7) \\ Notice that to “move” `1` to the other side of the equation we used inverses, associativity, and identity.

The Euclidean algorithm (or guess-and-check) can be used to show that \[ (4)(2) + (-1)(7) = 1. \] Thus, `2^{-1} \equiv 4 \ (\text{mod } 7)`.

Multiplying both sides by `2^{-1}`:

(4)(2x) &\equiv 4(4) \ (\text{mod } 7) \\ (4 \cdot 2)x &\equiv 16 \ (\text{mod } 7) \\ 8x &\equiv 16 \ (\text{mod } 7) \\ x &\equiv 2 \ (\text{mod } 7) \\
